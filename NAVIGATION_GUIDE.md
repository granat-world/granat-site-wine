# Navigation Guide

> Quick context for this repository in the broader Granat ecosystem.

## What this repository is

This is one of the **Cone Red Phase 1 supporting deliverables** in the `granat-world` organization — partner-facing sites, the manifesto, and design prototypes built alongside the core platform during Phase 1.

## Phase 1 acceptance status

This repository was **delivered to the Customer as-is** at the close of Phase 1. It is **outside the scope of Phase 2** and **outside the scope of the independent code review** that follows the Phase 1 acceptance act.

Cone Red provides no continuing maintenance commitment on this repository unless the Customer specifically directs that one of the supporting repositories enter active Phase 2 development.

## Where to start the actual code review

The two repositories in scope for the Phase 1 independent code review are:

- **`granat-world/GRANAT`** — backend choir-synthesis pipeline + FastAPI service
- **`granat-world/reactjs-miniapp-granat`** — Telegram Mini App

The master review guide — listing all 13 repositories in the org, their attribution (Cone Red core, Cone Red supporting, Evacode predecessor), and the recommended review path — lives at:

**[`granat-world/GRANAT/REVIEW_GUIDE.md`](https://github.com/granat-world/GRANAT/blob/main/REVIEW_GUIDE.md)**

Start there.
