# Navigation Guide

> Quick context for this repository in the broader Granat ecosystem.

## What this repository is

This is one of the **Cone Red goodwill artefacts** in the `granat-world` organization — work performed by Cone Red beyond the contracted Phase 1 scope, as a gesture of goodwill to support the Customer's broader vision (partner-facing surfaces, manifesto sites, and design explorations).

## Phase 1 acceptance status

This repository was **delivered to the Customer as-is** at the close of Phase 1. It is **outside the scope of Phase 2** and **outside the scope of the independent code review** that follows the Phase 1 acceptance act.

Cone Red provides no continuing maintenance commitment on this repository unless the Customer specifically directs that one of the goodwill repositories enter active Phase 2 development.

## Where to start the actual code review

The two repositories in scope for the Phase 1 independent code review are:

- **`granat-world/GRANAT`** — backend choir-synthesis pipeline + FastAPI service
- **`granat-world/reactjs-miniapp-granat`** — Telegram Mini App

The master review guide — listing **all 13** repositories in the org, their attribution (Cone Red vs Evacode predecessor team), and the recommended review path — lives at:
